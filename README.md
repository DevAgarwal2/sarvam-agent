# Sarvam Agent - Hermes Instance 1

A production-ready Hermes Agent Docker instance powered by Sarvam AI (chat, TTS, STT, OCR) with Mistral vision, Odoo ERP integration, browser automation, and email.

## Quick Start

```bash
git clone https://github.com/DevAgarwal2/sarvam-agent.git
cd sarvam-agent
cp .env.example .env   # edit with your API keys
docker compose up -d hermes-1
```

See [README.md](README.md) for full documentation.
